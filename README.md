# InvestInfo (bolsa verde-azul)

**Getting Started**

To run this project locally, ensure you have Node.js and npm installed.

1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`

## Technologies Used

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

You can deploy this project using any standard static hosting service (Vercel, Netlify, Cloudflare Pages, etc.) by running the build command:

`npm run build`
